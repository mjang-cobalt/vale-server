Vale offers multiple options for installation, each of which best suits a particular use case. The
quickest way to get started, though, is by using either
[Homebrew](https://formulae.brew.sh/formula/vale#default) (macOS and Linux),
[Chocolatey](https://chocolatey.org/packages/vale) (Windows), or
[Docker](https://hub.docker.com/r/jdkato/vale) (any):

:::info Heads up!
If you're using Vale with a markup format other than **Markdown** or **HTML**, you'll also need to
install a [parser](/vale/scoping#formats).
:::

## Using Vale with a continuous integration \(CI\) service

If you want to use Vale with a CI service, we recommend that you use the
official [GitHub CLI tool][1] to download the version of your choice from the
[releases page][2]

```bash
$ gh release download --repo 'errata-ai/vale' -p 'vale_2.11.2_Linux_64-bit.tar.gz'
$ tar -xf 'vale_2.11.2_Linux_64-bit.tar.gz'
$ ./vale -v
```

This will ensure that your CI builds are consistent and allow you to update
versions on an opt-in basis.

[1]: https://cli.github.com/
[2]: https://github.com/errata-ai/vale/releases
